En mathématiques, un polynôme est une expression constitée à partir de variables (ou indéterminées) et de coefficients, qui n'utilise pour opérations que l'addition, la soustraction, la multiplication et des exposants entiers naturels.

Il vous est donné une expression mathématique avec des parenthèses et une variable, comme chaîne de caractères (string). L'expression contient "x", des parenthèses et les opérateurs suivants : + - *. Il vous faut la convertir en polynôme. Les puissances de "x" seront représentées par "**".

Le polynôme de réponse sera présenté au format suivant: 

     CN*x**n+CN1*x**(n-1)+...+C1*x+C0

où
СN, СN1 ...С1, С0 - sont des coefficients de type entier (int).
Si C = 0, alors ne pas montrer ce terme (ex. 2*x**2+3). La seule exception est zero tout seul (ex. "x-x" == "0").
Si C = 1, alors ne pas montrer C. (ex. x**2+2*x). Seulement si ce n'est pas С0.
Si C = -1, alors ne pas montrer C, seulement "-". (ex. x*x-x). Seulement si ce n'est pas С0.
Si l'exposant de x est 1, alors ne pas montrer l'exposant (ex. 5*x).

Les termes doivent être triés par ordre décroissant, classés selon l'exposant de la variable.

Input: Une expression (string) (unicode).

Output: Un polynôme (string) sans espace.

Exemple:

simplify("(x-1)*(x+1)") == "x**2-1"
simplify("(x+1)*(x+1)") == "x**2+2*x+1"
simplify("(x+3)*x*2-x*x") == "x**2+6*x"
simplify("x+x*x+x*x*x") == "x**3+x**2+x"
simplify("(2*x+3)*2-x+x*x*x*x") == "x**4+3*x+6"
simplify("x*x-(x-1)*(x+1)-1") == "0"
simplify("5-5-x") == "-x"
simplify("x*x*x-x*x*x-1") == "-1"

Application: C'est une application très utile pour les logiciels de calcul formel mathématique, ou simplement pour faciliter vos devoirs de maths.

Préconditions:
0 < len(expr)
