You are given a mathematical expression with brackets and with one variable as a string (ex. "x*x*x + (x-1)*(x+1)"). You should convert this to a Polynomial. (Read more about this subject at Wikipedia). You can only use the following operators: + - *. Power of "x" should be represented by multiplication.

The final form of the polynoomial must be in the following format:
Ð¡N*x*x*...(n times)...*x+CN1*x*x*...(n-1 times)...*x+...+C1*x+C0
where
CN, CN1 ...C1, C0 - constants, integers.
if C = 0, then don't show this term (ex. 2*x*x+3)
if C = 1, then don't show C. (ex. x*x+2*x).
if C = -1, then don't show C, only "-". (ex. x*x-x).
Terms must be in descend order, sorted by the power of the variable.

Input: An expression as a string (unicode string).

Output: A polynomial as a string without whitespaces.

Example:

checkio(u"x*x*x+(x-1)*(x+1)") == "x*x*x+x*x-1"
checkio("(x+1)*(x+1)") == "x*x+2*x+1"
checkio("(x+3)*x*2-x*x") == "x*x+6*x"
checkio("x+x*x+x*x*x") == "x*x*x+x*x+x"
checkio("(2*x+3)*2-x+x*x*x*x") == "x*x*x*x+3*x+6"

How it is used: It very useful function for mathematical software.
